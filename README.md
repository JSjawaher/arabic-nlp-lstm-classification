# Arabic Text Classification using LSTM 🤖

## 📌 وصف المشروع
مشروع ذكاء اصطناعي باستخدام تقنيات معالجة اللغة الطبيعية (NLP) ونموذج LSTM لتصنيف النصوص العربية إلى أكثر من فئة في نفس الوقت (Multi-Label Classification).

---

## 🧠 التقنيات المستخدمة
- Python  
- TensorFlow / Keras  
- LSTM  
- Tokenizer  
- MultiLabelBinarizer  
- NumPy  

---

## 📊 خطوات العمل
- تجهيز النصوص العربية  
- تحويل النصوص إلى تسلسلات رقمية  
- ترميز التصنيفات (Multi-Label Encoding)  
- بناء نموذج LSTM  
- تدريب النموذج  
- التنبؤ بالتصنيفات  

---

## 🎯 الهدف من المشروع
فهم النصوص العربية وتصنيفها إلى أكثر من فئة باستخدام تقنيات التعلم العميق.

---

## 🚀 ملاحظات
تم تطوير المشروع لأغراض تعليمية في مجال الذكاء الاصطناعي ومعالجة اللغة الطبيعية.
